# PycPong
A recreation of the classic arcade game Pong built using Python and Pygame.

This project is part of a collection of game recreation made by me in order to learn about game development, game programming and game design. 
Every mechanic was implemented from scratch.

## Features

- Vertical (Y-Axis) movement.
- Two players gameplay.
- Collision detection.
- Progressive difficulty increase.
- Score system.

## What I learned and Improved

- Object-Oriented Programming (OOP).
- 2D Movement and player controls.
- Collision detection, clamp positions and bounces.
- Basic UI implementation.
- Time-based difficulty progression.

## Technologies
- Python 3.
- Pygame 2.

## Controls
### Player 1.
- W: Move up.
- S: Move down.

### Player 2
- ↑: Move up.
- ↓: Move down.

## Installation
```
git clone https://github.com/lautttdev/PycPong.git
cd PythonInvaders
pip install pygame
python main.py
```
